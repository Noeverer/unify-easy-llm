# unify-easy-llm（ULM）

本项目基于[Firefly](https://github.com/yangjianxin1/Firefly)修改旨在打造一个简易的一键式大模型训练工具，支持Nvidia GPU、Ascend NPU等不同硬件以及常用的大模型。


## 微调方法

- 全量微调
- LoRA微调

## 支持的模型

- baichuan2-7b/13b
- qwen-7b/14b
- qwen1.5-7b/14b

## 支持的硬件

- Nvidia A800、H800、RTX 3090、RTX 4090
- Ascend 910A、910B
